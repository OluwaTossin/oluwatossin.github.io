About Me

Cloud & Security Engineer | DevOps & MLOps Specialist | Problem Solver with a Systems Mindset

My journey into cloud engineering began with a simple question: How can we build systems that not only run, but scale securely and adapt under pressure?

Since then, I’ve spent the last few years helping startups, nonprofits, and tech companies answer that question by designing resilient infrastructures across AWS, GCP, and hybrid environments. I’ve delivered multi-region architectures, automated CI/CD pipelines, and secure ML platforms, achieving outcomes like 30% cost savings, 50% faster deployments, 70% audit-effort reduction, and 99.99% uptime.

The Tools I Use to Solve Problems

  • Infrastructure as Code: Terraform, CloudFormation, Ansible
  • CI/CD Automation: GitHub Actions, Jenkins, AWS CodePipeline
  • Observability: Prometheus, Grafana, CloudWatch (logs, metrics, alerts)
  • Security & Compliance: IAM (least privilege), KMS, AWS WAF, AWS Config, GuardDuty, Security Hub, SCPs / Access Analyzer
  • DevSecOps Tooling: Policy-as-Code (OPA, Checkov), SAST/SCA (CodeQL, Trivy, Snyk), Secrets Mgmt (SSM, Secrets Manager), XDR/SIEM (Sophos XDR/MDR)
  • MLOps Delivery: SageMaker, Step Functions, Lambda, Bedrock (secure endpoints, encrypted artifacts)
